# Dr. Alok Tiwari — Portfolio Redesign

A polished, single-page academic and professional portfolio optimized for GitHub Pages.

## Highlights
- Responsive single-page design
- Modern academic-tech visual language
- Publications filter
- Contact form integration via Formspree
- Fast static deployment

## Run locally
Open `index.html` directly in a browser, or serve it with a simple static server.

### Python
```bash
python3 -m http.server 8000
```
Then visit `http://localhost:8000`

## Deploy on GitHub Pages
1. Push these files to the root of your GitHub repository.
2. In GitHub, go to **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and `/root` folder.
5. Save and wait for the Pages URL to be published.
